実践演習＆応用

実践でのTIPS

反復的な改善 (Iterative Improvement)

  • まず生成: 最初から完璧を求めず、まずはAIに作らせてみる。
  • 評価・フィードバック: 生成結果を見て、どこが良いか、どこを修正したいか考える。
  • プロンプト修正: フィードバックを元に、プロンプトを具体化・修正して再生成。
  • 直接編集: AIの生成結果を元に、自分で加筆・修正する。
  • このサイクルを繰り返すことで、望む結果に近づけていく。

ツールの使い分け

  • 得意分野を活かす:
    • 長文生成や対話 → ChatGPT, Claude
    • 情報検索連携 → Gemini, Copilot
    • 芸術的な画像 → Midjourney
    • 自由度の高い画像生成 → Stable Diffusion
    • 指示忠実度の高い画像 → DALL-E 3
    • 商用利用・安全 → Adobe Firefly
  • 連携させる:
    • ChatGPTでアイデア出し → Stable Diffusionで画像化
    • Geminiで情報収集 → Claudeで長文レポート作成
  • コスト意識: 無料枠、有料プラン、API利用など、目的に応じてコスト効率の良いツールを選ぶ。
  • 色々試してみる: 実際に使ってみて、自分に合ったツールを見つけることが重要。
